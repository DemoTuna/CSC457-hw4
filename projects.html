Selected Projects

A selection of academic and personal projects demonstrating my skills and interests.


SALMA - Babysitting And Childcare Service Platform



SALMA is a conceptual mobile application designed to support working parents by providing a safe, flexible, and reliable babysitting service. The platform allows parents to search for babysitters, view detailed profiles, book sessions, and complete payments securely.

The project focuses on user trust, safety, and simplicity, inspired by real emotional experiences and the long-lasting impact of caregivers in children’s lives. The system was designed with verified identities, clear workflows, and scalable architecture to ensure reliability and future expansion.

This project includes full system analysis, UML diagrams, UI design, and architectural planning, demonstrating both technical and user-centered design skills.


Completed:April 2025



Seam Carving Using three Algorithmic Approaches



Seam Carving is an image processing project that focuses on content-aware image resizing using algorithm design techniques. The goal of the project is to intelligently reduce image dimensions while preserving important visual content.

The project explores and compares three different algorithmic approaches—brute force, dynamic programming, and greedy algorithms—to identify and remove low-energy seams from images. Through this comparison, the project demonstrates how algorithm choice impacts efficiency, accuracy, and performance in solving computational problems.


Completed:April 2025
